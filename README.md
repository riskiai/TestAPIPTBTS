#CARA INSTALL PROJECT DI GITHUB UNTUK API :
1. Pertama masuk ke github yang ingin di git clone
2. Kedua lakukan php artisan key:generate
4. Keempat melakukan php artisan migrate 
5. Lakukan php artisan serve
